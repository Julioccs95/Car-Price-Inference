# Car-Price-Inference

In this project, I analyzed car pricing data to identify the factors associated with vehicle prices using inferential statistical methods. The project included data cleaning, exploratory data analysis, statistical modeling, and interpretation of results. Multiple linear regression and coefficient hypothesis testing were used to determine which variables had the greatest influence on vehicle pricing.




## Skills demonstrated
- Data Cleaning
- Exploratory Data Analysis
- Regression modeling
- Statistical Inference
- model interpretation
- Business focused reporting
